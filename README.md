# Knot

* Android version >= 8.0
* Bookkeeping
* Counters
* Todo list
* Pedometer
* Statistical charts
* Statistical reports
* Reader(txt,epub)
* Notes(Markdown)
* Data export and import
* Backup data to OneDrive
