# Knot

* Android version >= 8.0
* Bookkeeping
* Counters
* Todo list
* Pedometer
* Statistical charts
* Statistical reports
* Reader(txt,epub,pdf)
* Notes(Markdown)
* Data export and import
* Backup data to OneDrive
* Clients(Win,Mac,Linux) can use [Syncthing](https://github.com/syncthing/syncthing) to seamlessly synchronize data.
