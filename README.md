# Knot

* Android version >= 6.0
* Bookkeeping
* Counters
* To-do list
* Pedometer
* Statistical charts
* Statistical reports
* Reader(txt,epub...)
* Memo
* Data export and import

### For data backup and synchronization, the following tools are available.
https://github.com/syncthing/syncthing
